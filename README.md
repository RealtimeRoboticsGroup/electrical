### 3. The OSHWA Logo (The "Gear")

If you want to signal that this is open-source hardware visually, it is common practice to add the **Open Source Hardware Logo** (a gear with a missing tooth) next to the license text. Most CAD libraries (like the SparkFun or Adafruit libraries) already have this logo as a "footprint" you can simply drop onto your board.
